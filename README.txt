# Dataset de Classificação de Papeleiras > 2023-04-24 11:28pm

Classificação de imagens | Tensorflow

Link para o Roboflow: https://universe.roboflow.com/object-detection/classificacao-de-papeleiras

Descrição do dataset: Este dataset é composto por imagens de lixeiras (chamadas tecnicamente de papeleiras) da cidade do Recife, que possuem o mesmo formato, porém são divididas conforme sua cor e adesivamento, compondo três classes de objetos. 
O objetivo do futuro treinamento será classificar imagens de lixeiras conforme suas características, visando orientar a tomada de decisão e validar a 
operação de manutenção destas, que é paga conforme o seu tipo.
